# 🧹Data-cleaning-ml

## 📌 Overview

This project focuses on cleaning and preprocessing a dataset for Machine Learning.

## 📂 Dataset
* Source:https://www.kaggle.com/datasets/souradippal/student-performance-prediction?resource=download
## ⚙️ Steps Performed

* Handling missing values
* Removing duplicates
* Outlier detection
* Data normalization / scaling
* Encoding categorical data

## 🛠️ Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn

## 📊 Output

* Cleaned dataset ready for ML models

## 🚀 How to Run

1. Install requirements:

```
pip install -r requirements.txt
```

2. Open the notebook:

```
jupyter notebook
```

## 💡 Key Learnings

* Importance of data cleaning
* Handling outliers
* Feature scaling techniques
